## 🎙️ Recording Tips for Mac

### Screen Recording (QuickTime or Cmd+Shift+5):
1. **QuickTime Method**:
   - Open QuickTime Player
   - File → New Screen Recording
   - Select "Entire Screen" or drag to select area
   - Click record, perform demo actions
   - Stop recording (Cmd+Control+Esc)

2. **Built-in Method** (macOS Mojave+):
   - Press Cmd+Shift+5
   - Select "Record Entire Screen" or "Record Selected Portion"
   - Click "Options" → Select microphone (if recording voiceover live)
   - Click "Record"

### Voiceover Recording:
1. **Option A: Record separately in Voice Memos**
   - Open Voice Memos app
   - Record voiceover reading script
   - Export as .m4a file
   - Import to iMovie

2. **Option B: Record directly in iMovie**
   - Select clip in timeline
   - Window → Record Voiceover (or press V)
   - Click red microphone button
   - Read script while watching video

### Audio Quality Tips:
- Use AirPods or external mic for better quality
- Record in quiet room (close windows, turn off AC)
- Speak clearly and at moderate pace
- Leave 1-second pause between sentences
- Do a test recording first

## 🎬 iMovie Step-by-Step Instructions

### 1. Create New Project
1. Open iMovie
2. File → New Movie
3. Select "No Theme"
4. Name: "Kisan Saarthi Demo - AI4Bharat 2026"

### 2. Import Media
1. File → Import Media
2. Import:
   - Title card image
   - Architecture diagrams (screenshots from mermaid.live)
   - Screen recordings (7 clips)
   - Voiceover audio (if recorded separately)
   - Background music (optional)

### 3. Arrange Timeline
1. Drag clips to timeline in order (Scene 1-7)
2. Trim clips to exact durations:
   - Scene 1: 10s
   - Scene 2: 25s
   - Scene 3: 20s
   - Scene 4: 45s
   - Scene 5: 25s
   - Scene 6: 30s
   - Scene 7: 25s

### 4. Add Transitions
1. Click "Transitions" tab
2. Drag "Cross Dissolve" between clips (1 second each)
3. Add "Fade In" at start (1 second)
4. Add "Fade Out" at end (2 seconds)

### 5. Add Text Overlays
1. Click "Titles" tab
2. Select "Lower Third" or "Centered" style
3. Drag to timeline above video
4. Edit text for each scene (see script above)
5. Adjust duration and position

### 6. Add Voiceover
**Option A: If recorded separately**
1. Drag audio file to timeline below video
2. Align with video clips
3. Adjust volume (right-click → Audio Adjustments)

**Option B: Record in iMovie**
1. Select clip in timeline
2. Window → Record Voiceover (or press V)
3. Click record and read script
4. Repeat for each scene

### 7. Add Background Music (Optional)
1. Click "Audio" tab
2. Select music from iMovie library or import your own
3. Drag to timeline below voiceover
4. Adjust volume to 10-15% (background only)
5. Add "Fade In" and "Fade Out" to music

### 8. Fine-Tune Audio
1. Select voiceover clip
2. Right-click → Audio Adjustments
3. Set volume to 100%
4. Enable "Reduce background noise" if needed
5. Select background music
6. Set volume to 10-15%

### 9. Add Callouts/Highlights (Optional)
1. Use "Picture in Picture" for zoom effects
2. Use "Split Screen" for comparisons
3. Add shapes/arrows using Keynote, export as PNG with transparency

### 10. Export Video
1. File → Share → File
2. Settings:
   - Resolution: 1080p
   - Quality: High
   - Compress: Better Quality
3. Save as: "Kisan_Saarthi_Demo_AI4Bharat_2026.mp4"
4. Upload to YouTube/Google Drive for submission
