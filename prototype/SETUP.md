# Quick Start Guide

## Installation

```bash
cd prototype
npm install
```

## Run Development Server

```bash
npm run dev
```

The app will open at `http://localhost:3000`

## Build for Production

```bash
npm run build
```

## Technology

- React 18 + Vite
- React Router DOM
- Recharts for data visualization
- Custom CSS styling

## Features

1. **Rural Farmer Dashboard** - Farm-level carbon tracking
2. **Emission Hotspot View** - BOM-level supply chain analysis  
3. **Cooperative View** - Aggregated FPO metrics

All data is mocked for demo purposes.
